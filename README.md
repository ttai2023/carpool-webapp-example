## 📖 Description

Triton Carpool is a student-built web application that allows UCSD students to:

- **Browse available rides** offered by fellow Tritons
- **Offer rides** and share their commute with others
- **Book seats** on existing rides with a single click
- **Filter & search** rides by destination or vehicle type
- **Toggle dark/light mode** for comfortable viewing
- **Connect** with the campus carpooling community

This project demonstrates core web development concepts including HTML structure, CSS styling with Flexbox, JavaScript DOM manipulation, event listeners, localStorage for data persistence, and scroll animations.

## 📁 Project Structure

```
triton-carpool/
├── index.html      # Main HTML file — page structure and content
├── styles.css      # CSS stylesheet — UCSD branding, layout, dark mode
├── script.js       # JavaScript — app logic, event handling, data storage
└── README.md       # Project documentation (you're reading it!)
```

- **index.html** — Contains all the HTML markup: navbar, header, about section, rides grid, ride offer form, connect form, resources, and footer.
- **styles.css** — All styling using UCSD's official brand colors (Navy `#182B49`, Blue `#006A96`, Gold `#FFCD00`). Includes CSS variables, Flexbox layouts, responsive design, dark mode, and scroll reveal animations.
- **script.js** — Application logic: ride CRUD operations using localStorage, dark mode toggle, form handling with `event.preventDefault()`, animated stat counters, search/filter, modal system, and scroll reveal.
